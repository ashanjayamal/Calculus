Chapter 5 Derivative and the Shape of the Graph

Definition 5.1 (Strictly Increasing and Strictly Decreasing Functions) The function \(f\) is strictly increasing on an interval \(I\) if \[f(x_1) < f(x_2) \text{ whenever } x_1, x_2 \in I \text{ and } x_1 < x_2\]

The function \(f\) is strictly decreasing on an interval \(I\) if \[f(x_1) > f(x_2) \text{ whenever } x_1, x_2 \in I \text{ and } x_1 < x_2\]

Theorem 5.1 (Monotone Function Theorem) Let \(f\) be differentiable on the open interval \((a,b)\). If \(f'(x) > 0\) for every \(x\) in \((a,b)\), then \(f\) is strictly increasing on \((a,b)\). If \(f'(x) < 0\) for every \(x\) in \((a,b)\), then \(f\) is strictly decreasing on \((a,b)\).
